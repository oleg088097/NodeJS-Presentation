First of all

Node is JavaScript for servers

Node allows you to quickly create network applications. Transferring data between the client and server is very rapid since NodeJS makes it possible to write JavaScript on both sides. If your use case does not involve intensive computational load on the CPU or access to blocked resources, you can safely take advantage of Node.js and enjoy fast and scalable network applications!

Hello World NodeJS http server

						
							const http = require('http');

							const hostname = '127.0.0.1';
							const port = 3000;

							const server = http.createServer((req, res) => {
							  res.statusCode = 200;
							  res.setHeader('Content-Type', 'text/plain');
							  res.end('Hello World\n');
							});

							server.listen(port, hostname, () => {
							  console.log(`Server running at http://${hostname}:${port}/`);
							});

What is Node besides web server?

NodeJS in a nutshell

  • Open source & cross-platform project
  • Build on Chrome's JavaScript runtime environment
  • Event driven & Non-blocking IO
  • Single threaded
  • Build in Async IO, HTTP/s, file-system etc
  • Ability to do JavaScript on server
  • Modules
